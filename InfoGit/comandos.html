  • Comandos
  • git init
    • Inicia un nuevo repositorio de Git en el directorio actual.
    • Crea una carpeta oculta llamada ".git" que almacena todos los datos y metadatos del repositorio.
  • git clone [URL]
    • Clona un repositorio Git existente desde una URL remota a tu sistema local.
    • Descarga una copia completa del repositorio, incluyendo todos los archivos y la historia de cambios.
  • git add [archivo(s)]
    • Agrega archivos al área de preparación (staging area) para que Git los incluya en el próximo commit.
    • Puedes especificar archivos individuales o usar "." para agregar todos los archivos modificados en el directorio actual.
  • git commit -m "[mensaje]"
    • Crea un nuevo commit con los archivos en el área de preparación.
    • El mensaje es una descripción breve que explica los cambios realizados en el commit.
  • git status
    • Muestra el estado actual de los archivos en el repositorio.
    • Te informa sobre los archivos modificados, los archivos en el área de preparación y otros detalles relevantes.

Ejemplo de clonar un repo en nuestro escritorio con una consola

            
                cd desktop
                git clone https://github.com/AngelMagaquian/AW1.git
                cd AW1
